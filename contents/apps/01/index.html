Coconut Library Tools

Coconut Library Tool app was developed by two librarians in order to help librarians or anyone who needs cool features but is hindered by programming languages. Using this app librarians can perform (i) keyword stem, (ii) topic modeling, (iii) bidirected network, and (iv) sunburst. This app works on Scopus’s CSV file, Web of Science’s Tab delimited file, and custom CSV file.

For performing topic modeling, the developers were inspired by Chapter-4 of the book.
