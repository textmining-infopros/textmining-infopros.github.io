About

Chapter 1: The Computational Library covers the case study on Clustering of Documents using Two Different Tools. It is divided into two parts: 1A, and 1B. 1A case study uses Orange tool whereas 1B case study uses R programming language to perform clustering. The RStudio and Jupyter Notebook are associated with 1B case study.

For corrections/suggestions reach me at lambamanika07@gmail.com or create an issue here.

Data

In addition to the provided sample data, you can use dataset from Appendix A, Appendix B, Appendix C, Curated Datasets, or your own dataset to perform clustering.

How To

Click on Interactive RStudio Server to open a virtual RStudio environment for hands-on practice for the case study. In the virtual environment, open the clustering.R file to run the code.

Similarly, you can use Jupyter Notebook to run the code for the case study. You can either use the Interactive Jupyter Notebook or the Static Jupyter Notebook
